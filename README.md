# 📦 Smart Inventory & Warehouse Management System

![Project Status](https://img.shields.io/badge/Project_Status-Active-green?style=for-the-badge&logo=git)
![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python)
![Flask](https://img.shields.io/badge/Framework-Flask-black?style=for-the-badge&logo=flask)

A comprehensive Full-Stack inventory management application built with **Python** and **Flask**. This system provides real-time stock tracking, financial planning tools, and data security, serving as a powerful tool for small to medium-sized enterprises.

## 🌟 Key Features

### 📈 Financial Control & Analytics
*   **Live Profit Calculation:** Dynamic real-time calculation of profit per unit during data entry.
*   **Inventory Valuation:** Automatic assessment of total warehouse value on the Dashboard, based on current stock levels and costs.

### ⚡ Smart Management (UX/UI)
*   **Auto-Fetch Mechanism:** Instant retrieval of item details (Name, Prices, Limits) via the Fetch API.
*   **Quick Updates:** One-click stock increment/decrement (+/-) buttons directly from the product list.
*   **Live Search:** Real-time product filtering as you type.
*   **Low Stock Alerts:** Visual color-coded indicators for products below safety thresholds.

### 🛡️ Security & Compliance
*   **User Authentication:** Secure Login/Register system with encrypted passwords (Password Hashing).
*   **Advanced Audit Log:** Full activity tracking (who changed what and when) for total transparency.
*   **Adaptive Dark Mode:** Theme switching with preference saving via browser cookies.

---

## 💻 Tech Stack

*   **Backend:** Python 3.10+, Flask
*   **Database:** SQLite3
*   **Frontend:** HTML5, CSS3 (Glassmorphism), Vanilla JavaScript (ES6+)
*   **Security:** Werkzeug Security Helpers

![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python)
![Flask](https://img.shields.io/badge/Framework-Flask-black?style=for-the-badge&logo=flask)

A comprehensive Full-Stack inventory management application built with **Python** and **Flask**. This system provides real-time stock tracking, financial planning tools, and data security, serving as a powerful tool for small to medium-sized enterprises.

## 🌟 Key Features

### 📈 Financial Control & Analytics
*   **Live Profit Calculation:** Dynamic real-time calculation of profit per unit during data entry.
*   **Inventory Valuation:** Automatic assessment of total warehouse value on the Dashboard, based on current stock levels and costs.

### ⚡ Smart Management (UX/UI)
*   **Auto-Fetch Mechanism:** Instant retrieval of item details (Name, Prices, Limits) via the Fetch API.
*   **Quick Updates:** One-click stock increment/decrement (+/-) buttons directly from the product list.
*   **Live Search:** Real-time product filtering as you type.
*   **Low Stock Alerts:** Visual color-coded indicators for products below safety thresholds.

### 🛡️ Security & Compliance
*   **User Authentication:** Secure Login/Register system with encrypted passwords (Password Hashing).
*   **Advanced Audit Log:** Full activity tracking (who changed what and when) for total transparency.
*   **Adaptive Dark Mode:** Theme switching with preference saving via browser cookies.

---

## 💻 Tech Stack

*   **Backend:** Python 3.10+, Flask
*   **Database:** SQLite3
*   **Frontend:** HTML5, CSS3 (Glassmorphism), Vanilla JavaScript (ES6+)
*   **Security:** Werkzeug Security Helpers

---

## ⚙️ Installation & Setup

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
    cd YOUR_REPO_NAME
    ```

2.  **Create a Virtual Environment (Optional but Recommended):**
    ```bash
    python -m venv venv
    # Activate on Windows:
    venv\Scripts\activate
    # Activate on Mac/Linux:
    source venv/bin/activate
    ```

3.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Run the Application:**
    ```bash
    python app.py
    ```
    The app will be available at: `[http://127.0.0.1:5000](http://127.0.0.1:5000)`

---

## 🛠️ Roadmap (Future Updates)
*   **📊 Data Visualization:** Integration of charts for monthly profit trends.
*   **📥 Export Tools:** Functionality to export reports to Excel and PDF.
*   **📧 Notifications:** Automated email alerts for critical stock levels.
