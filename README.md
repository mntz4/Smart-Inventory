# 📦 Smart Inventory & Warehouse Management System

![Project Status](https://img.shields.io/badge/Project_Status-Active/Developing-green?style=for-the-badge&logo=git)
![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python)
![Flask](https://img.shields.io/badge/Framework-Flask-black?style=for-the-badge&logo=flask)

A comprehensive Full-Stack inventory management application. This system provides real-time stock tracking, financial planning tools, and data security, serving as a powerful tool for the digital transformation of small to medium-sized enterprises.

## 🌟 Key Features

### 📈 Financial Control & Analytics
*   **Live Profit Calculation:** Dynamic real-time calculation of profit per unit during data entry.
*   **Inventory Valuation:** Automatic assessment of total warehouse value on the Dashboard, based on current stock levels and purchase costs.

### ⚡ Smart Management (UX/UI)
*   **Auto-Fetch Mechanism:** Instant retrieval of item details (Name, Prices, Limits) upon entering the SKU/Barcode using the Fetch API.
*   **Quick Inventory Updates:** One-click stock increment/decrement (+/-) buttons directly from the product list.
*   **Live Search:** Real-time product filtering as you type.
*   **Low Stock Alerts:** Visual color-coded indicators for products that fall below safety thresholds.

### 🛡️ Security & Compliance
*   **User Authentication:** Secure Login/Register system with encrypted passwords (Password Hashing).
*   **Advanced Audit Log:** Full activity tracking (who changed what and when) for total transparency.
*   **Adaptive Dark Mode:** Theme switching with automatic preference saving via browser cookies.

---

## 🛠️ Roadmap (Active Development)
The project is continuously being upgraded. Upcoming features include:

*   **📊 Data Visualization:** Integration of charts to monitor monthly profit trends and stock movements.
*   **📥 Export Tools:** Functionality to export warehouse reports to Excel and PDF formats.
*   **📧 Smart Notifications:** Automated email alerts when stock levels are critical.
*   **📱 Mobile PWA:** Optimization as a Progressive Web App for seamless mobile use.
